# Modern Weather Dashboard

## Overview

Modern Weather Dashboard is an advanced desktop-based weather application built using Python and Tkinter.  
It retrieves real-time weather data from WeatherAPI and presents it through a modern, user-friendly graphical interface.

The project demonstrates API integration, multithreading, GUI design, data visualization, and robust error handling.

---

## Features

- Graphical user interface built with Tkinter  
- Real-time current weather information  
- Hourly weather forecast (next 12 hours)  
- Extended 5-day weather forecast  
- Temperature unit conversion (Celsius and Fahrenheit)  
- Wind speed, humidity, pressure, visibility, and cloud data  
- UV index categorization  
- Air Quality Index interpretation  
- Active weather alerts and warnings  
- Temperature trend visualization using Matplotlib  
- Multithreaded data fetching for smooth performance  
- Input validation and error handling  

---

## Technologies Used

- Python 3
- Tkinter (GUI development)
- WeatherAPI (Weather data provider)
- Requests (HTTP requests)
- Matplotlib (Data visualization)
- Threading (Background API calls)
